# COLAB-Parent-Teacher-Data
Dataset and supporting materials for the COLAB parent-teacher engagement study. This repository contains supplementary materials, anonymized datasets, questionnaires, qualitative and quantitative data used in the study:
“Collaborative Online Learning (COLAB) User Interface Guidelines to Enhance Parent-Teacher Engagement in High Power Distance Cultures”.

## Contents

- Parent and teacher questionnaire datasets
- Hofstede VSM 2013 questionnaire
- Parent questionnaire instrument
- Teacher questionnaire instrument
- WhatsApp focus group discussion (FGD) chat logs for parents and teachers
- Supporting supplementary materials

## Ethical Considerations

All participant information has been anonymized to protect privacy and confidentiality. Personal identifiers such as names, phone numbers, school names, and email addresses have been removed or replaced with participant codes.

All informed consent forms were obtained from the participants prior to data collection.

## Notes

The datasets and supplementary materials are provided for academic and research purposes only.
